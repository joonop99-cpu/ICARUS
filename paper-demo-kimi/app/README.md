# App launcher (macOS)

## 1) Install app bundle
```bash
cd paper-demo-kimi
bash app/install_macos_app.sh
```

Creates:
- `~/Applications/PaperOrchestrator.app`

## 2) Run without app (CLI)
```bash
bash app/run_gui.sh
```

GUI URL:
- <http://localhost:8502>

## 3) Stop
```bash
bash app/stop_gui.sh
```

## Logs
- `paper-demo-kimi/.logs/gui.log`
